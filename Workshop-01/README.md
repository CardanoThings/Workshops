# Workshop 01: The Basics

This workshop starts with the very basics to set you up for the coming workshops: You will install and fund a Cardano wallet on the Preprod Network, install the Arduino IDE, set up your Microcontroller and make your first API call.

## Workshop Documentation

[View full workshop on cardanothings.io](https://cardanothings.io/workshops/01-basics)

## Steps

1. **Cardano Setup** - Install and setup a Cardano Wallet like Yoroi for the Preprod Testnet
2. **Arduino Setup** - Learn the basics of Arduino Script, connect your Microcontroller to your PC and WiFi
3. **API Setup & First Call** - Fetch data like the current Epoch Number from an API like Koios

## Examples

This folder contains the following code examples:

- `blink-code` - Basic LED blink example
- `wifi-code` - WiFi connection example
- `koios-api-code` - First API call to Koios

## Requirements

**Hardware:**
- ESP8266 or ESP32 based Microcontroller
- Micro USB Cable
- Computer with Arduino IDE installed
- WiFi Network with Internet Access

**Software:**
- Webbrowser
- Yoroi Wallet (Preprod Testnet)
- Arduino IDE
